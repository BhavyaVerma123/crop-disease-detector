# 🌾 Crop Disease Detector

A Machine Learning-powered web application built to help farmers and researchers identify and classify crop diseases instantly from uploaded images. 

---

## 🚀 Features
* **Instant Detection:** (The project is still underway)
* **Flask Web Interface:** A lightweight, responsive front-end designed for easy user interaction.
* **Actionable Insights:** Provides details about the detected disease to help take timely preventive measures.

## 🛠️ Tech Stack
* **Language:** Python
* **Framework:** Flask
* **Machine Learning:** TensorFlow / PyTorch / Scikit-Learn
* **Deployment:** Production-ready via `Procfile`

## 💻 Installation & Local Setup

Follow these simple steps to get the project running locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/holdmy-awm/crop-disease-detector.git](https://github.com/holdmy-awm/crop-disease-detector.git)
   cd crop-disease-detector
Install the dependencies:
```bash
pip install -r requirements.txt
```
Run the application:
```bash
python app.py
```
Access the app:
Open your browser and navigate to http://127.0.0.1:5000/
